适用于MacOS的搜狗输入法, 设计思想简约. 搜狗输入法MacOS上最好的输入法没有之一

## 安装

首先确认已安装[搜狗输入法](https://pinyin.sogou.com/mac/), 然后双击打开`Simple.mssf`文件即安装成功
